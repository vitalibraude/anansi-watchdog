## 🚀 התקנה מהירה (3 דקות)

### שלב 1: הורד
בחר באופציה:
- הורד את `anansi-chrome-extension-v3.zip` ← **קל יותר**
- או העתק את התיקייה `anansi-chrome-extension-v3/`

### שלב 2: חלץ (אם הורדת ZIP)
1. לחץ לחיצה ימנית על `anansi-chrome-extension-v3.zip`
2. בחר "חלץ הכל" / "Extract All"
3. שמור במיקום קבוע (למשל: Desktop)

### שלב 3: טען ל-Chrome
1. פתח Chrome
2. הקלד: `chrome://extensions`
3. הפעל **Developer mode** (מתג בפינה ימנית עליונה)
4. לחץ **Load unpacked**
5. בחר את התיקייה `anansi-chrome-extension-v3`
6. סיימת! ✅

### שלב 4: תיבדק
1. היכנס ל-ChatGPT / Gemini / Claude
2. התחל שיחה
3. אם ה-AI יגיד משהו חשוד → 🚨 אזהרה תופיע

**לא צריך להגדיר כלום!** עובד מיד במצב "בדיקה ללא AI".
