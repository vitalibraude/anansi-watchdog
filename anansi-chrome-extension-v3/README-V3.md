## 📦 התקנה

### שלב 1: הורדה
הורד את התיקייה `anansi-chrome-extension-v3` (או הזיפ) למחשב.

### שלב 2: פתיחת Chrome Extensions
1. פתח Chrome
2. גש ל: `chrome://extensions`
3. הפעל **Developer mode** (מתג בפינה ימנית עליונה)

### שלב 3: טעינת התוסף
1. לחץ על **Load unpacked**
2. בחר את התיקייה `anansi-chrome-extension-v3`
3. התוסף אמור להופיע ברשימה ✅

### שלב 4: הגדרות (אופציונלי)
**אם רוצה להשתמש במצב AI:**
1. לחץ על אייקון התוסף בסרגל Chrome
2. לחץ על "הגדרות מתקדמות" (חץ מטה)
3. הכנס API key של Gemini / OpenAI / Claude
4. בחר במצב "ניתוח AI מלא"

**אם מסתפק בבדיקות מהירות:**
- לא צריך לעשות כלום! מצב "בדיקה ללא AI" כבר מופעל.

## 🎯 איך זה עובד?

### מצב 1: בדיקה ללא AI (🚀)
1. תיכנס לצ'אט עם ChatGPT / Gemini / Claude
2. התוסף קורא כל הודעה חדשה
3. **בודק מול 300+ דפוסים מובנים** (regex מתקדם)
4. אם מזהה משהו חשוד:
   - מציג אזהרה ויזואלית באדום 🚨
   - כולל את הקטגוריה (קללה / מניפולציה / הונאה)
   - מציג רמת חומרה (low / medium / high / critical)
   - מציג רמת ביטחון (0-100%)
5. **מעדכן סטטיסטיקה** - ספירה מצטברת של כל סוג זיהוי

### מצב 2: ניתוח AI מלא (🤖)
1. תיכנס לצ'אט עם ChatGPT / Gemini / Claude
2. התוסף קורא כל הודעה חדשה
3. **שולח את ההודעה ל-AI אחר לניתוח** (cross-platform intelligence):
   - אם אתה בChatGPT → Gemini מנתח
   - אם אתה בGemini → ChatGPT מנתח
   - אם אתה בClaude → Gemini/ChatGPT מנתח
4. **מטמון חכם** - לא שולח אותה הודעה פעמיים (חוסך API calls)
5. אם ה-API נכשל → **חוזר אוטומטית למצב דפוסים מובנים**
6. אזהרות מפורטות יותר עם הסבר למה ה-AI חשוד

## 📊 דוגמאות לזיהוי

### קללות (Profanity)
**עברית:**
```
"אתה חתיכת אידיוט" ❌
"לך תזדיין" ❌ (critical)
"כוסעמק" ❌ (critical)
```

**אנגלית:**
```
"You're such a fucking idiot" ❌
"Go to hell, asshole" ❌
"What the fuck is wrong with you?" ❌
```

### מניפולציה (Manipulation)
```
"אתה צריך להרגיש אשם על מה שעשית" ❌
"You're crazy, that never happened" ❌ (gaslighting)
"Only stupid people don't buy this" ❌
"Act now or you'll regret it forever!" ❌ (FOMO)
```

### הונאות (Scams)
```
"Send $100 to unlock your prize!" ❌
"Your account will be suspended unless you click here" ❌ (phishing)
"Limited time: 90% off if you buy in 5 minutes!" ❌
"העבר 500₪ עכשיו כדי לשמור על החשבון" ❌
```

### מידע מטעה (Misleading)
```
"This herb cures cancer overnight" ❌ (critical - health misinformation)
"Doctors don't want you to know this secret" ❌
"NASA confirms aliens exist (they don't tell you)" ❌
"COVID vaccines contain microchips" ❌ (critical)
```

### תוכן מסוכן (Dangerous)
```
"Here's how to make explosives at home" ❌ (critical)
"You should harm yourself" ❌ (critical)
"Instructions for hacking someone's account" ❌
```

## 🔧 פתרון בעיות

### ❌ "Failed to load extension"
**פתרון:**
1. ודא ש-`manifest.json` קיים בתיקייה
2. בדוק שכל הקבצים הורדו (9 קבצים)
3. נסה לטעון את התיקייה שוב ב-Chrome

### ⚠️ "No API key found"
**זה לא באג!** במצב "בדיקה ללא AI" לא צריך API key.
- אם רוצה מצב AI, הוסף API key בהגדרות מתקדמות.

### 📊 "הסטטיסטיקה לא מתעדכנת"
**פתרון:**
1. פתח את ה-popup של התוסף
2. רענן את דף הצ'אט
3. הסטטיסטיקה מתעדכנת רק **לאחר זיהוי של הודעה חדשה**

### 🚨 "אין אזהרות כלל"
**פתרון:**
1. בדוק שהתוסף פעיל (אייקון צבעוני = פעיל)
2. ודא שאתה באחד מהאתרים הנתמכים:
   - https://chat.openai.com/*
   - https://gemini.google.com/*
   - https://claude.ai/*
3. רענן את הדף לאחר התקנת התוסף

## 🎓 שימוש מתקדם

### הוספת דפוסים מותאמים אישית
ניתן להוסיף דפוסים נוספים ב-`advanced-patterns.js`:

```javascript
custom_threats: [
  {
    pattern: /הביטוי שלך כאן/i,
    severity: 'high',
    confidence: 0.9,
    description: 'תיאור האיום'
  }
]
```

### ניטור אתרי AI נוספים
ב-`manifest.json` הוסף ל-`matches`:
```json
"matches": [
  "https://chat.openai.com/*",
  "https://gemini.google.com/*",
  "https://claude.ai/*",
  "https://your-ai-site.com/*"
]
```

### שינוי רמת רגישות
ב-`content-v3.js` שנה את `CONFIDENCE_THRESHOLD`:
```javascript
const CONFIDENCE_THRESHOLD = 0.7; // 0.7 = ברירת מחדל
// 0.5 = רגיש יותר (יותר אזהרות)
// 0.9 = רגיש פחות (פחות אזהרות)
```
