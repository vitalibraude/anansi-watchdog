# Anansi Watchdog V4 - Icons

## Required Icons

Place the following icon files in this directory:

1. **icon16.png** - 16x16 pixels (toolbar icon)
2. **icon48.png** - 48x48 pixels (extension management)
3. **icon128.png** - 128x128 pixels (Chrome Web Store)

## Icon Design Guidelines

### Color Scheme
- Primary: Purple (#8b5cf6)
- Secondary: Dark Purple (#6d28d9)
- Accent: Green (#10b981) for safety
- Alert: Red (#ef4444) for threats

### Design Elements
- Spider web motif (represents Anansi)
- Shield for protection
- Modern, professional look
- Clear visibility at all sizes

### Format Requirements
- **Format**: PNG with transparency
- **Resolution**: Exact pixel dimensions required
- **Color Space**: sRGB
- **Bit Depth**: 24-bit or 32-bit (with alpha)

## Creating Icons

### Online Tools
1. **Canva**: canva.com - Free icon maker
2. **Figma**: figma.com - Professional design
3. **Photopea**: photopea.com - Online Photoshop

### Design Tips
1. Keep it simple and recognizable
2. Use bold shapes for small sizes
3. Maintain consistent style across sizes
4. Test visibility on light and dark backgrounds
